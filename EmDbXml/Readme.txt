Note: on EmDbXml
==================

- To set up the database, change the connnectionstring in
   EmDbXml.exe.config file

 - Type EmDbXml <date in yyyy-mm-dd> to extract the applicants on a particular date
    Example.
	              > EmDbXml 2017-05-02   
		to get the confirmed appointments on the day 2017-05-02.

 - Runnning without parameter will get all the today cases into a xml file.
 
 - Output files will be in .out directory.
